# Performance-des-ventes-de-formations

 Dashboard de ventes de formations – Projet Analyse de Données
 Présentation du projet
Ce projet consiste en un tableau de bord analytique réalisé avec Microsoft Excel, conçu pour suivre et analyser la performance des ventes de formations sur plusieurs années.
Le dashboard transforme des données brutes en indicateurs clés de performance (KPI) et en visualisations dynamiques afin de faciliter la prise de décision basée sur les données.

 Objectifs métier
•	Analyser l’évolution annuelle des ventes de formations
•	Évaluer la performance du chiffre d’affaires par formation
•	Identifier les formations les plus et les moins performantes
•	Aider à la prise de décisions stratégiques grâce à la visualisation des données

 Indicateurs clés (KPI)
•	Chiffre d’affaires total par année
•	Nombre de formations vendues
•	Chiffre d’affaires par formation
•	Répartition des ventes par année et par formation
•	Performance globale des ventes

 Outils & méthodes
•	Microsoft Excel
•	Nettoyage et structuration des données
•	Tableaux croisés dynamiques
•	Graphiques dynamiques
•	Segments pour le filtrage interactif
•	Bonnes pratiques de conception de dashboards

  Valeur analytique
Ce dashboard permet :
•	Une vue globale et synthétique de la performance des ventes
•	Une analyse des tendances (hausse / baisse)
•	Une comparaison efficace entre les différentes formations
•	Une base solide pour une future migration vers des outils BI (Power BI, Tableau)

 Public cible
•	Data Analysts junior
•	Business Analysts
•	Responsables de formation
•	Toute personne souhaitant apprendre l’analyse de données et le reporting avec Excel

 Améliorations futures
•	Intégration avec Power BI
•	Actualisation automatique des données
•	Ajout de KPI avancés (taux de croissance, contribution par formation)
